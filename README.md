
# CS1.6 服务器构建平台
CS 1.6 服务器构建平台基于HLDS，包含ReHLDS、ReGameDLL等

## 01.HLDS
* 官方发布的Windows和Linux版本

## 02.ReHLDS
* 官方发布的3.15.0.896以及3.15.0.898-DEV版本

## 03.ReGameDLL
* regamedll-bin-5.30.0.814.zip - 官方发布的 5.30.0.814
* regamedll-bin-5.30.0.814-CN+ShowC4_D.kit.zip - 参数汉化，增加mp_scoreboard_showc4_dkit参数，死亡CT可看到T持有C4，死亡T可看到CT持有拆弹器.

## 04.Reunion
* 提供0.2.0.25、0.2.0.34、0.2.0.36三个版本，其中：reunion-0.2.0.36-cn.zip为汉化版本

## 05.Custom
* 汉化BotChatter.db（zbot聊天信息）、BotProfile.db（自定义zbot），zbot.cfg等参数设置文件
* 完整的zbot音效、模型、AI人质等文件
* 已修改的liblist.gam文件
* 汉化版本的Rehlds.CFG、Server.CFG.

## 06.MetaMod

## 07.Amxmodx
